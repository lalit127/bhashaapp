Download Nunito from Google Fonts:
https://fonts.google.com/specimen/Nunito

Required weights:
- Nunito-Regular.ttf     (400)
- Nunito-Light.ttf      (500)
- Nunito-SemiBold.ttf    (600)
- Nunito-Bold.ttf        (700)
- Nunito-ExtraBold.ttf   (800)
- Nunito-Black.ttf       (900)

Why Nunito:
- Rounded letterforms = friendly, modern (like Duolingo)
- Excellent support for Devanagari fallback
- High legibility at small sizes
- Free, open source (SIL OFL license)
